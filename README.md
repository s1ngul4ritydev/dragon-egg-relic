# DragonEgg Relic

A Minecraft Bedrock 26.1.2 addon that transforms the Dragon Egg into a unique relic.

Players carrying the Dragon Egg receive powerful bonuses, making it one of the most valuable items in the world. The relic cannot be dropped manually and only changes ownership when its holder dies.

## Features

* Strength II while carrying the Dragon Egg
* Bonus health for the relic holder
* Dragon Egg cannot be dropped manually
* Relic drops on death
* Automatic transfer of effects to the next holder
* Multiplayer friendly
* Lightweight and optimized
* Vanilla-friendly gameplay

## How It Works

1. Obtain the Dragon Egg.
2. Keep it in your inventory.
3. Receive Strength II and bonus health automatically.
4. If you die, the Dragon Egg is dropped.
5. Another player can claim the relic and receive its effects.

## Compatibility

* Minecraft Bedrock 26.1.2
* Multiplayer Supported
* SMP Friendly

## Installation

1. Download the latest release.
2. Import the `.mcaddon` file into Minecraft Bedrock Edition.
3. Enable the addon in your world.
4. Start playing.
